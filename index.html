CSS3 动画

Transform, Transition, Animation

Version 1.0.0

前言

随着网络的发展，浏览器具有更强的渲染更高级代码的能力。 本文着重介绍变形(transform)、转换(transition)和动画(animation)CSS3动画技术。

transform

实现了一些可用SVG实现的同样的功能，可用于内联(inline)元素和块级(block)元素。它允许我们旋转、缩放和移动元素而只需要一行CSS码代。

阅读详细>>

transition

涉及到CSS属性的基本的转换。就是定义和移动一个元素从它的静止状态到它的hover或者激活状态，转换可以和变形同时使用以创建一些动画。

阅读详细>>

animation

可以定义多个关键帧以及定义每个关键帧中元素的属性值来实现复杂的动画效果。相对于使用脚本实现的动画，其拥有更加高效的性能。

阅读详细>>

Transform

实现了一些可用SVG实现的同样的功能，可用于内联(inline)元素和块级(block)元素，允许旋转、缩放和移动元素。

transform-origin

设置或检索对象以某个原点进行转换

该属性提供2个参数值

如果提供两个，第一个用于横坐标，第二个用于纵坐标

如果只提供一个，该值将用于横坐标；纵坐标将默认为50%

对应的脚本特性为transformOrigin

transform-origin：[<percentage>|<length>left|right] [<percentage>|<length>top|bottom]

默认值：50% 50%，效果等同于center center

通过设置不同的transform-origin值，观察对相同角度转向rotate(45deg)的影响。

   -moz-transform-origin: [percentage|length] [percentage|length];
    -ms-transform-origin: [percentage|length] [percentage|length];
        transform-origin: [percentage|length] [percentage|length];

scale

若数值为负，则翻转并按数值的绝对值相应缩放

CSS3 Transition

定义和移动一个元素从它的静止状态到它的hover或者激活状态，转换可以和变形同时使用以创建一些动画。

transition

复合属性。检索或设置对象变换时的过渡。

可以为同一元素的多个属性定义过渡效果。

对应的脚本特性为transition。

transition：[ transition-property ] || [ transition-duration ] || [ transition-timing-function ] || [ transition-delay ]

background-color, border-color, color
width, height, line-height
margin-left, margin-right, margin-top, margin-bottom
padding-left
opacity
font-size, font-weight
/* 设置参与过渡的属性为 all, 持续时间为 .5s, 运动算法为 ease-in-out,延迟过渡的时间为 0. */
-webkit-transition: all .5s ease-in-out;
   -moz-transition: all .5s ease-in-out;
    -ms-transition: all .5s ease-in-out;
        transition: all .5s ease-in-out;

property

检索或设置对象中的参与过渡的属性。

如果提供多个属性值，以逗号进行分隔。

对应的脚本特性为transitionProperty。

默认值为：all。默认为所有可以进行过渡的css属性。

transition-property：all | none | <property>[ ,<property> ]*

background-color, border-color, color
width, height, line-height

 line-height, margin, padding, opacity, font-size, font-weight */
-webkit-transition-property: color, opacity, width, padding, background-color; //省略
   -moz-transition-property: color, opacity, width, padding, background-color; //省略
    -ms-transition-property: color, opacity, width, padding, background-color; //省略
        transition-property: color, opacity, width, padding, background-color; //省略

duration

检索或设置对象过渡的持续时间。

如果提供多个属性值，以逗号进行分隔。

对应的脚本特性为transitionDuration。

默认值：0。

timing function

检索或设置对象中过渡的动画类型。

如果提供多个属性值，以逗号进行分隔。

对应的脚本特性为transitionTimingFunction。

linear：线性过渡。等同于贝塞尔曲线(0.0, 0.0, 1.0, 1.0)

ease：平滑过渡。等同于贝塞尔曲线(0.25, 0.1, 0.25, 1.0)

ease-in：由慢到快。等同于贝塞尔曲线(0.42, 0, 1.0, 1.0)

ease-out：由快到慢。等同于贝塞尔曲线(0, 0, 0.58, 1.0)

ease-in-out：由慢到快再到慢。等同于贝塞尔曲线(0.42, 0, 0.58, 1.0)

cubic-bezier(<number>, <number>, <number>, <number>)：特定的贝塞尔曲线类型，4个数值需在[0, 1]区间内

默认值：ease。

transition-timing-function：linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(<number>, <number>, <number>, <number>)[ ,linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(>number<, <number>, <number>, <number>) ]*

通过设置相同的transition-duration, transition-delay，观察不同transition-timing-function的效果。

(0, 0, 0.58, 1.0)，(0.42, 0, 0.58, 1.0)，(0.0, 0.0, 1.0, 1.0)  */
-webkit-transition-timing-function: ease, ease-in, ease-out, ease-in-out, linear;
  -moz-transition-timing-function: ease, ease-in, ease-out, ease-in-out, linear;
   -ms-transition-timing-function: ease, ease-in, ease-out, ease-in-out, linear;
       transition-timing-function: ease, ease-in, ease-out, ease-in-out, linear;

direction

检索或设置对象动画在循环中是否反向运动。

注意当动画处于反向运动时，对应的运动类型也会翻转。例如使用ease-in会变成ease-out。

对应的脚本特性为animationDirection。

normal：默认值，正常方向。

reverse：动画反向运行,方向始终与normal相反。

alternate：动画会循环正反方向交替运动，奇数次会正常运动，偶数次会反向运动，即所有相关联的值都会反向。

alternate-reverse：动画从反向开始，再正反方向交替运动，运动方向始终与alternate定义的相反。

animation-direction: normal | reverse | alternate | alternate-reverse [, normal | reverse | alternate | alternate-reverse ]*

JS Events

使用js来给动画元素添加指定的类，动画本身定义在css，它避免了脚本里到处是硬编码的css，实现分离逻辑。

animation event

Animation 可以监听三个事件，分别是：animationstart、animationend、animationiteration

事件均支持冒泡，且可撤销。

animation

animation start

事件在动画开始时触发。设置animation-delay时，触发在延迟之后。

上下文信息：animationName

//for firefox and IE10
element.addEventListener("animationstart", function(e) {
            //do something
       }
//for chrome and safari
element.addEventListener("wekitAnimationStart", function(e) {
            //do something
       }

animation end

事件在动画结束后触发。

上下文信息：animationName,elapsedTime

//for firefox and IE10
element.addEventListener("animationend", function(e) {
            //do something
       }
//for chrome and safari
element.addEventListener("wekitAnimationEnd", function(e) {
            //do something
       }

animation iteration

事件在animation-iteration-count大于1时才触发。

上下文信息：animationName,elapsedTime

//for firefox and IE10
element.addEventListener("animationinteration", function(e) {
            //do something
       }
//for chrome and safari
element.addEventListener("wekitAnimationInteration", function(e) {
            //do something
       }
